<a id="sobre"></a>
## Sobre
O **Gerenciador de Orçamentos para Freelancers** foi desenvolvido para resolver a desorganização no fluxo de propostas comerciais. Através de uma interface limpa e intuitiva, a ferramenta permite que o usuário gerencie clientes e gere orçamentos profissionais de forma rápida e eficiente.

---

<a id="funcionalidades"></a>
## Funcionalidades
- [ ] **Automação:** Geração de documentos com um clique.
- [ ] **Integração com IA:** Sugestão inteligente de preços e prazos.
- [ ] **Tema e acessibilidade:** Interface adaptável com foco em produtividade.
- [ ] **Gestão de Dados:** Cadastro e edição de informações em tempo real.
